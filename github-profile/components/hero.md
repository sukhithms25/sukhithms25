# Component Design Specification: Hero (The Identity Layer)

This specification details the layout, typography, visual style, and contents of the Hero section.

---

## Component 1 — Banner
* **Decision:** Custom OS Terminal Banner (SVG image).
* **Details:** A simulated operating system/terminal terminal window header with macOS style window controls (red, yellow, green window actions) and a center title reading `[SPECTER: CORE-OS v0.1.0-alpha]`. The background color will match our Deep Space Black (`#0b0e14`).

---

## Component 2 — Name
* **Decision:** `Sukhith M S`
* **Details:** Bold title size, clean browser-native sans-serif font. No emojis.

---

## Component 3 — Identity
* **Decision:** `AI Systems Engineer`
* **Details:** Displayed immediately under the name. Framed in a terminal console command style (e.g. `guest@specter:~$ whoami -> AI Systems Engineer`).
